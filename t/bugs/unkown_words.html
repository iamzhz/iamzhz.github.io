但是我想用 UTF-8 编码啊？！

搜索“Windows gcc utf-8”，emmmmmm，只要在编译时加上 -finput-charset=UTF-8 -fexec-charset=GBK，就可以啦。于是我的 makefile 变成这样了 (不过我不会把它 push 到 Github 上)

cc := g++
std := -std=c++11
abandon: src/*.cc src/*/*.cc
	$(cc) $^ -o $@ $(std) -g -finput-charset=UTF-8 -fexec-charset=GBK
